## 色板

```css
:root {
  /* 基底 */
  --felt-900: #142B22;   /* 最深底色（页面边缘渐晕） */
  --felt-700: #1B3B2F;   /* 牌桌毛氈 · 主背景 */
  --felt-500: #26523F;   /* 盘面格子底/分隔 */
  /* 牌 */
  --ivory-100: #FBF7EC;  /* 牌面高光 */
  --ivory-300: #F5EFE0;  /* 牌面主色 */
  --ivory-500: #E4D9C0;  /* 牌面阴影/侧边 */
  /* 强调 */
  --gold-300: #E3C98A;   /* 金 · 高光 */
  --gold-500: #C9A45C;   /* 金 · 边框/分隔/赢奖金额 */
  --gold-700: #9A7B3F;   /* 金 · 暗部 */
  --cinnabar-500: #B33A3A; /* 朱砂红 · 中/萬字/胡牌横幅 */
  --cinnabar-700: #8E2C2C;
  --bamboo-500: #3E7C59;   /* 竹绿 · 發/條子 */
  --indigo-500: #35507A;   /* 黛蓝 · 風牌（東南西北）字色 */
  --ink-900: #2B2A26;      /* 墨 · 正文/描边 */
}
```

用色纪律：金色只给「价值」（边框、金额、金牌、赢奖）；朱砂红只给「强调/庆祝」；大面积永远是墨绿+象牙白。禁止霓虹发光色。

## 字体

| 用途 | 字体 | 备注 |
|---|---|---|
| 牌面字/横幅大字 | **霞鹜文楷**（LXGW WenKai，OFL 开源） | 楷体书法感；子集化仅嵌入用到的字（中發東南西北萬筒條 + 数字 + 碰杠胡自摸天胡等 ≈40 字） |
| UI 数字/金额 | 系统等宽栈 `ui-monospace` | 金额滚动不跳宽 |
| admin 正文 | 系统 UI 栈 | 不用书法体做正文 |

## 麻将牌规格

- 牌面比例 **宽:高 = 3:4**，圆角半径 = 宽 × 0.12
- 结构：象牙白牌面（顶部微高光渐变）+ 底部 8% 高度的侧边条（`--ivory-500`，营造立体厚度）+ 1px 金描边（`--gold-700`，透明度 40%）
- 三态：
  - **正常**：如上
  - **中奖高亮**：金色外发光（柔和，radius = 宽×0.15）+ 牌面提亮 5%
  - **消除（打出）**：缩放 1.0→1.15→0，金箔粒子 8–12 颗迸出，180ms
- 特殊牌：白板 = 空白牌面 + 淡金云纹；骰子 = 象牙白底红点骰面；金牌 = 金底（`--gold-500` 渐变）+ 朱砂倍数字

## 五档赢奖演出规格

| 档 | 阈值(×bet) | 演出 | 时长 |
|---|---|---|---|
| 碰 | ≥5 | 牌面高亮 + 小字「碰」角标 + 金额滚动 | 0.8s |
| 杠 | ≥10 | 「杠」中字 + 金箔粒子一阵 | 1.2s |
| 胡了！ | ≥25 | 书法「胡了！」压屏 + 红绸带横扫 + 金箔雨 | 2.0s |
| 自摸 | ≥50 | 「自摸」压屏 + 全盘金光 + 绸带×2 | 2.5s |
| 天胡 | ≥100 | 「天胡」大字 + 满屏金箔雨 + 屏幕微震 + 金额慢滚 | 3.5s+ |

所有演出可点击跳过（金额直接到账显示）。

## 动效时长基准

落牌 320ms/列（列间错峰 60ms）· 消除 180ms · 下落补位 260ms（ease-out-back 轻回弹）· 近失慢停时该列拉长至 1.6s。Turbo 模式全部 ×0.4。

## 布局

- 桌面：盘面居中，最大 720px 宽；下注栏固定底部
- 移动竖屏：盘面近正方形铺满宽度，上方状态栏（余额/倍数/进度条），下方大 spin 按钮（拇指区）
- admin：左侧导航（看板/配置/玩家/审计），内容区卡片式；沿用色板但以浅色为主（象牙白底 + 墨绿导航）

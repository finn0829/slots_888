# WEB 前端设计审计（V1 → 商业级差距分析）

> 2026-07-14 · 方法：frontend-design 技能视角 + Playwright 实机截图（静置/连锁/桌面）
> 对标参照：PG Soft《Mahjong Ways》、Pragmatic《Sweet Bonanza》的表现层密度

## 总评

V1 的骨架是对的（国风方向、牌面几何、消除下落流程都成立），但它现在读起来是一个**干净的原型**，不是一台**老虎机**。商业 slots 和我们的差距不在"画得更好看"，而在三个系统性的缺失：

1. **场景包裹感（Scene）**：商业机的盘面"长在"一个世界里——有环境、有景深、有画框。我们的盘面漂在一块纯色渐变上。
2. **演出密度（Celebration Density）**：商业机每一次交互都有 3~5 层反馈叠加（粒子+音效+缩放+光效+数字滚动）。我们每个事件只有 1 层。
3. **材质（Material）**：商业机的每个元素都有质感——牌是玉的、字是烫金的、按钮是雕出来的。我们的元素是"填色 + 描边"。

## 逐层审计

### 1. 场景与空间 ⚠️ 最大短板

- 背景是一个 radial-gradient，**没有任何内容**。CT-4 里定义的"淡墨山水/祥纹暗纹"没有落地。
- 盘面没有画框。牌直接铺在毛毡色上，缺一圈"雕花木框/描金边"把它变成一张**实体牌桌**。
- **空间浪费严重**：手机竖屏上盘面上方一大块空白（flex 居中的副作用）；1440 桌面上两侧各 ~440px 纯空白，游戏缩在中间一条。
- 零环境氛围：没有缓慢飘落的金箔/光尘，没有晕影（vignette）聚焦视线。

### 2. 牌面与材质 ⚠️

- 牌面结构（象牙渐变+侧边厚度+圆角）成立，但**没有材质**：无高光扫过、无微噪点、字无凹刻/烫金感。
- **字体是最大的品牌风险**：CSS 里引用了霞鹜文楷但**从未加载**，实际渲染完全取决于用户系统（Linux 落到 AR PL UKai，Windows 落到楷体，无字库的机器直接衬线体）。牌面字就是这个游戏的脸，必须内嵌子集化 woff2（仅 ~40 字，几十 KB）。
- 白板（Wild）是"文字+方框"，是全盘最弱的一张——它应该是最贵气的（云纹浮雕/流光边）。
- 金牌/骰子没有区别于普通牌的"稀有感"（无呼吸光、无入场特写）。

### 3. 动效手感 ⚠️

已有：列错峰落牌、消除缩放、补位回弹。缺的是让手感"值钱"的那些：

- **落地无重量**：牌落定没有 squash & stretch（压扁回弹）、没有细微尘土/震动，像贴图平移。
- **消除无能量**：中奖牌只是缩小消失。应该是"打出去"——爆出 8~12 颗金箔粒子（CT-4 已定义规格，未实现）。
- **赢奖数字不滚动**：金额瞬间跳变。商业机的金额滚动（ease-out 计数）是多巴胺的一半。
- 连锁倍数是个小角标；商业机的倍数是**主角**（大字弹出、飞向金额）。
- 无近失 anticipation（骰子差一个时的慢停拉长）——需求池 WEB-6，未做。
- 完全无声。音效（WEB-7）对手感的贡献 ≥ 视觉。

### 4. 演出层级 ⚠️

- 五档横幅目前 = 半透明黑底 + 大字。CT-4 定义的**金箔雨、红绸带横扫、书法压屏动画**一项都没实现。
- 各档差异只有字号和时长，没有"档次感"的物理差异（天胡应该满屏金雨+微震屏）。

### 5. UI Chrome

- 按钮是通用圆角矩形；spin 按钮没有"主角感"（商业机的 spin 键是全屏最精致的物件）。
- Turbo 是个 emoji ⚡。保底进度条太素——它是我们的差异化机制，值得做成"骰盅/积分算珠"的具象物。
- 顶栏"雀·胡"字号小、无匾额/印章感，品牌存在感弱。

### 6. 质量地板

- 无 `:focus-visible` 焦点样式；未响应 `prefers-reduced-motion`；Canvas 无 aria-label。
- DPR 适配 ✓、60fps ✓（当前绘制量下有充足余量给粒子系统）。

## 改造方案（按 ROI 排序）

**签名元素定调**：这台机器被记住的应该是——"**一张真实的牌桌 + 打出去的牌爆成金箔**"。所有投入围绕它。

| 优先 | 项 | 内容 | 对应需求池 |
|---|---|---|---|
| P0-1 | 内嵌书法字体 | 霞鹜文楷子集化 woff2（牌面字+横幅字 ~40 字），Canvas 用 FontFace 加载后再画 | DES-1 |
| P0-2 | 场景层 | 暗纹背景 + 晕影 + 盘面描金木框 + 顶部匾额式 logo + 慢速金箔环境粒子 | DES-2 |
| P0-3 | 中奖演出重做 | 消除=金箔粒子爆 + 金额 ease-out 滚动 + 倍数大字弹出；五档横幅按 CT-4 落地（金雨/红绸/微震屏） | WEB-4 |
| P0-4 | 落地手感 | squash & stretch + 列落地错峰微震 | WEB-2 打磨 |
| P1-1 | 近失 anticipation | 3 骰子时未停列慢停+高亮+紧张光效 | WEB-6 |
| P1-2 | 音效 | Web Audio 合成：落牌"啪"、消除、金箔、各档赢奖、免费旋转 BGM | WEB-7 |
| P1-3 | 布局重构 | 竖屏消灭空白（盘面上移+顶部做场景）；桌面利用两侧（赔付表/近期赢奖侧栏） | WEB-9 |
| P1-4 | Wild/金牌/骰子特权化 | 呼吸光、入场特写、专属材质 | DES-1 |
| P2 | 焦点样式 / reduced-motion / aria | 质量地板 | — |

预计 P0 四项做完，"demo 感 → 产品感"的转变就能完成 70%；P1 补完手感闭环。
